## English
Are you tired of missing key bindings in the help menu (F1) just because you have too many mods installed or are using complex vehicles? This mod solves the problem by adding a pagination system to the input help display. If there are more entries than can fit on the screen, you can simply scroll through multiple pages.

Features:
- Adds pages to the F1 help menu if lines exceed the limit
 - Displays the current page number (e.g., "Page 1 / 2") directly in the help window
- Works on foot and inside vehicles
- Key bindings are fully configurable in the game options

Console commands:
- ihpNext => go to the next page in the help menu
- ihpPrev => go to the previous page in the help menu
- ihpDump => output actions in the help menu in the console (for transfer to the config file)
- ihpReload => reload the config file while the game is running

Default Controls:
- Page Up: Previous Page
- Page Down: Next Page

## Deutsch

Bist du es leid, dass Tastenbelegungen im Hilfemenü (F1) abgeschnitten werden, nur weil du viele Mods installiert hast oder komplexe Fahrzeuge bedienst? Dieser Mod löst das Problem, indem er eine Blätter-Funktion (Pagination) zur Eingabehilfe hinzufügt. Wenn mehr Einträge vorhanden sind, als auf den Bildschirm passen, kannst du einfach durch mehrere Seiten blättern.

Funktionen:
- Fügt dem F1-Hilfemenü Seiten hinzu, wenn das Limit an Zeilen überschritten wird
- Zeigt die aktuelle Seitenzahl (z. B. "Seite 1 / 2") direkt im Hilfefenster an
- Funktioniert zu Fuß und in Fahrzeugen
- Tastenbelegung ist in den Spieloptionen frei konfigurierbar

Konsolenbefehle:
- ihpNext => zur nächsten Seite wechseln im Hilfemenü
- ihpPrev => zur vorherigen Seite wechseln im Hilfemenü
- ihpDump => Aktionen im Hilfemenü in der Konsole ausgeben (zur Übernahme in die Config Datei)
- ihpReload => Neuladen der Config Datei im laufenden Spiel

Standard-Steuerung:
- Bild Auf: Vorherige Seite
- Bild Ab: Nächste Seite
